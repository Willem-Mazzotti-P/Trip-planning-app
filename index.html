Mobile Road Trip Guide

Open the EV itinerary on your phone like a small app.

This page is the phone-friendly launch point for the rendered Quarto guide. Once hosted, the full route map and charging plan can be opened in the browser or installed to the home screen.

Open full guide
Publish this `docs/` folder on GitHub Pages or another static host, then open that URL on the phone.

What this gives you

  • A website entry point that works well on a smartphone.
  • An installable web app manifest for supported browsers.
  • Offline-friendly caching for the landing page and recently opened route guide files.

How to use it

  • Host the contents of `docs/`.
  • Open the site URL on the phone.
  • Use the install prompt or the browser menu item such as "Add to Home Screen".
